# Temp Email Tester

A Next.js web app for testing your site's registration forms with temporary email addresses and automated code extraction.

## Quick Start

1. **Read the design:** `DESIGN.md` — understand the architecture
2. **Follow the plan:** `PLAN.md` — step-by-step implementation guide
3. **Get your Mailtrap Account ID:**
   - Log into [Mailtrap](https://mailtrap.io)
   - Go to Settings > API Tokens
   - Copy your Account ID
4. **Run `PLAN.md` Phase 1-5** to build and deploy

## Features

- ✅ Generate unique temp emails on demand
- ✅ View incoming emails in real-time (auto-refreshes every 5s)
- ✅ Extract verification codes (`###-###` format) automatically
- ✅ One-click copy code to clipboard
- ✅ Reset to new email and clear inbox
- ✅ Live on Vercel in 10 minutes
